# Portfolio | Alex Cavalcante Costa

Portfólio pessoal desenvolvido para apresentar trajetória profissional, habilidades técnicas, projetos e informações de contato de Alex Cavalcante Costa — profissional em transição para a área de TI com foco em dados e segurança da informação.

---

## 🎯 Objetivo

Este projeto foi desenvolvido para apresentar minha trajetória profissional, habilidades técnicas e projetos desenvolvidos durante minha transição para a área de Tecnologia da Informação, com foco em desenvolvimento e segurança da informação.

---

## 📌 Status do Projeto

✅ Em desenvolvimento contínuo

Este projeto recebe melhorias constantes como parte da minha evolução profissional na área de Tecnologia da Informação.

## ✨ Funcionalidades

- Navegação com destaque automático da seção ativa via `IntersectionObserver`
- Seção de apresentação (hero) com tags de especialidade e chamadas para ação
- Linha do tempo de experiência profissional
- Listagem de formação acadêmica e certificações
- Grade de habilidades técnicas
- Card de projeto em destaque com link externo
- Seção de contato com e-mail, telefone, LinkedIn e link para projeto
- Download de currículo em PDF diretamente pelo navegador
- Layout responsivo com tema escuro

## ▶️ Como executar localmente

1. Clone o repositório para sua máquina.
2. Abra a pasta do projeto no **VS Code**.
3. Execute o projeto utilizando a extensão **Live Server**, ou abra o arquivo `index.html` diretamente no navegador.

> Nenhuma dependência ou instalação adicional é necessária.
